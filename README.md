# Scripter
Scripter is a wrapper around Google's v8 javascript engine
- The Goal of the project is for easy access to a javascript engine with an easier API

## Platforms
|  Platform  |  Status     |
|------------|-------------|
|  Linux     |  Supported  |
|  Win32     |  Not yet    |
|  MacOS     |  Maybe      |
|  Android   |  Not yet    |
